CodeinTanglish

CodeinTanglish is a beginner-friendly programming learning platform designed for Tamil speakers who are new to coding. The platform teaches programming concepts using Tanglish (Tamil written using English letters) so beginners can easily understand technical topics without struggling with complex English terminology.

The goal of this project is to make coding accessible, simple, and less intimidating for Tamil beginners who want to start their journey in software development.

🚀 Features

Beginner-friendly coding lessons

Programming concepts explained in simple Tanglish

Step-by-step tutorials for easy understanding

Practical coding examples

AI-powered assistance using Gemini API

Clean and responsive UI for better learning experience

🎯 Mission

Many Tamil students want to learn programming but face difficulty because most educational resources are available only in English.

CodeinTanglish aims to bridge this gap by explaining programming concepts in a way Tamil learners naturally understand.

Our mission is to:

Make programming accessible to Tamil beginners

Reduce fear and confusion around coding

Encourage more Tamil students to enter the tech field

Build a supportive Tamil developer community

🧠 Topics Covered

CodeinTanglish focuses on foundational programming concepts such as:

Programming basics

Variables and data types

Conditional statements

Loops

Functions

Basic problem solving

Introduction to web development

More tutorials and topics will be added in future updates.

📚 Example Learning Style

Concepts are explained in a beginner-friendly Tanglish format.

Variable na enna?

Variable na oru box mathiri.
Andha box la data store pannalam.

Example:
let age = 20;

This approach helps beginners connect programming concepts with familiar language.

🛠 Tech Stack

CodeinTanglish is built using modern web technologies:

React – Frontend framework

Tailwind CSS – Styling and UI design

Gemini API – AI-powered explanation system
