<h1 align="center">CodeinTanglish</h1>

<p align="center">
A beginner-friendly coding platform for Tamil learners using Tanglish.
</p>

<hr>

<h2>🚀 About the Project</h2>

<p>
<strong>CodeinTanglish</strong> is a beginner-friendly programming learning platform designed for
Tamil speakers who are new to coding. The platform explains programming concepts using
<strong>Tanglish (Tamil written using English letters)</strong> so beginners can understand
technical topics without struggling with complex English terminology.
</p>

<p>
The goal of this project is to make coding <strong>accessible, simple, and less intimidating</strong>
for Tamil beginners starting their journey into software development.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
<li>Beginner-friendly coding lessons</li>
<li>Programming explained in simple Tanglish</li>
<li>Step-by-step tutorials</li>
<li>Real coding examples</li>
<li>AI-powered explanations using Gemini API</li>
<li>Clean and responsive UI</li>
</ul>

<hr>

<h2>🎯 Mission</h2>

<p>
Many Tamil students want to learn programming but face difficulties because most
resources are available only in English.
</p>

<p>
<strong>CodeinTanglish aims to bridge this gap</strong> by explaining programming concepts in
a way Tamil learners naturally understand.
</p>

<ul>
<li>Make programming accessible for Tamil beginners</li>
<li>Reduce fear and confusion around coding</li>
<li>Encourage Tamil students to enter the tech industry</li>
<li>Build a strong Tamil developer community</li>
</ul>

<h2>📚 Example Learning Style</h2>

<pre>
Variable na enna?

Variable na oru box mathiri.
Andha box la data store pannalam.

Example:
let age = 20;
</pre>

<p>
This approach helps beginners connect programming concepts with familiar language.
</p>
